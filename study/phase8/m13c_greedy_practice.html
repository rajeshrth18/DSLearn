Assign Cookies — LeetCode #455 Easy
Pattern: Sort + Two Pointer · O(n log n)

Assign cookies to children: child i has greed factor g[i]; cookie j has size s[j]. Assign each child at most one cookie s[j] >= g[i]. Maximize the number of content children.

Sort both arrays. Use two pointers: try to satisfy the least greedy child with the smallest available cookie.
C# 
public int FindContentChildren(int[] g, int[] s) {
    Array.Sort(g); Array.Sort(s);
    int i = 0, j = 0;
    while (i < g.Length && j < s.Length) {
        if (s[j] >= g[i]) i++;
        j++;
    }
    return i;
}
Python 
def findContentChildren(g, s):
    g.sort(); s.sort()
    i = j = 0
    while i < len(g) and j < len(s):
        if s[j] >= g[i]: i += 1
        j += 1
    return i

Jump Game — LeetCode #55 Medium
Pattern: Greedy max reach · O(n)

Given nums[i] = max jump length at position i, return true if you can reach the last index.

Track the farthest reachable index. If current index exceeds it, return false.
C# 
public bool CanJump(int[] nums) {
    int reach = 0;
    for (int i = 0; i < nums.Length; i++) {
        if (i > reach) return false;
        reach = Math.Max(reach, i + nums[i]);
    }
    return true;
}
Python 
def canJump(nums):
    reach = 0
    for i, v in enumerate(nums):
        if i > reach: return False
        reach = max(reach, i + v)
    return True

Gas Station — LeetCode #134 Medium
Pattern: Greedy total + local sum · O(n)

There are n gas stations on a circuit. gas[i] is fuel available, cost[i] to travel to next. Find starting index to complete the circuit, or -1 if impossible.

If sum(gas) < sum(cost), return -1. Otherwise, a solution always exists. Track cumulative net gas; whenever it drops below 0, reset start to i+1.
C# 
public int CanCompleteCircuit(int[] gas, int[] cost) {
    int total = 0, tank = 0, start = 0;
    for (int i = 0; i < gas.Length; i++) {
        int diff = gas[i] - cost[i];
        total += diff; tank += diff;
        if (tank < 0) { start = i + 1; tank = 0; }
    }
    return total >= 0 ? start : -1;
}
Python 
def canCompleteCircuit(gas, cost):
    total = tank = start = 0
    for i, (g, c) in enumerate(zip(gas, cost)):
        total += g - c; tank += g - c
        if tank < 0: start = i + 1; tank = 0
    return start if total >= 0 else -1

Candy — LeetCode #135 Hard
Pattern: Two-pass greedy · O(n)

Each child in a line has a rating. Give every child at least 1 candy; children with higher rating than their neighbor get more. Minimize total candies.

Two passes: (1) Left-to-right: if rating[i] > rating[i-1], candy[i] = candy[i-1]+1. (2) Right-to-left: if rating[i] > rating[i+1], candy[i] = max(candy[i], candy[i+1]+1).
C# 
public int Candy(int[] r) {
    int n = r.Length;
    var c = new int[n]; Array.Fill(c, 1);
    for (int i = 1; i < n; i++)
        if (r[i] > r[i-1]) c[i] = c[i-1] + 1;
    for (int i = n-2; i >= 0; i--)
        if (r[i] > r[i+1]) c[i] = Math.Max(c[i], c[i+1]+1);
    return c.Sum();
}
Python 
def candy(ratings):
    n = len(ratings); c = [1] * n
    for i in range(1, n):
        if ratings[i] > ratings[i-1]: c[i] = c[i-1] + 1
    for i in range(n-2, -1, -1):
        if ratings[i] > ratings[i+1]: c[i] = max(c[i], c[i+1]+1)
    return sum(c)

Minimum Number of Arrows to Burst Balloons — LeetCode #452 Medium
Pattern: Sort by end, greedy grouping · O(n log n)

Balloons span [xstart, xend] on a wall. One arrow shot at position x bursts all balloons with xstart = x = xend. Minimum arrows to burst all balloons?

Sort by end. Shoot arrow at the end of the first balloon. It bursts all overlapping balloons. When next balloon starts after this arrow position, shoot a new arrow.
C# 
public int FindMinArrowShots(int[][] pts) {
    Array.Sort(pts, (a, b) => a[1].CompareTo(b[1]));
    int arrows = 1; long pos = pts[0][1];
    foreach (var p in pts)
        if (p[0] > pos) { arrows++; pos = p[1]; }
    return arrows;
}
Python 
def findMinArrowShots(points):
    points.sort(key=lambda x: x[1])
    arrows, pos = 1, points[0][1]
    for s, e in points:
        if s > pos: arrows += 1; pos = e
    return arrows

Partition Labels — LeetCode #763 Medium
Pattern: Last occurrence + greedy extend · O(n)

Partition string s into as many parts as possible so each letter appears in at most one part. Return list of partition sizes.

Record last occurrence of each character. Scan: extend current partition's end to max(end, last[s[i]]). When i == end, partition is complete.
C# 
public IList<int> PartitionLabels(string s) {
    var last = new int[26];
    for (int i = 0; i < s.Length; i++) last[s[i]-'a'] = i;
    var res = new List<int>();
    int start = 0, end = 0;
    for (int i = 0; i < s.Length; i++) {
        end = Math.Max(end, last[s[i]-'a']);
        if (i == end) { res.Add(end - start + 1); start = i + 1; }
    }
    return res;
}
Python 
def partitionLabels(s):
    last = {c: i for i, c in enumerate(s)}
    res, start, end = [], 0, 0
    for i, c in enumerate(s):
        end = max(end, last[c])
        if i == end: res.append(end - start + 1); start = i + 1
    return res

Best Time to Buy and Sell Stock II — LeetCode #122 Medium
Pattern: Collect all positive differences · O(n)

You can buy/sell stock multiple times (no holding). Maximize profit.

Collect every positive day-to-day increase: profit += max(0, prices[i] - prices[i-1]). This simulates buying and selling on every upward step.
C# 
public int MaxProfit(int[] p) {
    int profit = 0;
    for (int i = 1; i < p.Length; i++)
        profit += Math.Max(0, p[i] - p[i-1]);
    return profit;
}
Python 
def maxProfit(prices):
    return sum(max(0, prices[i]-prices[i-1]) for i in range(1, len(prices)))

Reorganize String — LeetCode #767 Medium
Pattern: MaxHeap + greedy interleave · O(n log k)

Rearrange characters of string s so no two adjacent characters are the same. Return rearranged string, or "" if impossible.

Impossible if max frequency > (n+1)/2. Otherwise, use a max-heap: at each step, pick the most frequent character not equal to last placed. Use count[-freq, char] trick.
C# 
public string ReorganizeString(string s) {
    var freq = new int[26];
    foreach (var c in s) freq[c-'a']++;
    var pq = new PriorityQueue<char, int>(Comparer<int>.Create((a,b) => b-a));
    for (int i = 0; i < 26; i++) if (freq[i] > 0) pq.Enqueue((char)('a'+i), -freq[i]);
    var sb = new System.Text.StringBuilder();
    while (pq.Count > 1) {
        pq.TryDequeue(out var c1, out var p1);
        pq.TryDequeue(out var c2, out var p2);
        sb.Append(c1); sb.Append(c2);
        if (-p1 - 1 > 0) pq.Enqueue(c1, p1+1);
        if (-p2 - 1 > 0) pq.Enqueue(c2, p2+1);
    }
    if (pq.Count == 1) { pq.TryDequeue(out var last, out var _); sb.Append(last); }
    return sb.Length == s.Length ? sb.ToString() : "";
}
Python 
from collections import Counter
import heapq
def reorganizeString(s):
    heap = [(-v, k) for k, v in Counter(s).items()]
    heapq.heapify(heap)
    res = []
    while len(heap) >= 2:
        c1, k1 = heapq.heappop(heap)
        c2, k2 = heapq.heappop(heap)
        res += [k1, k2]
        if -c1 - 1 > 0: heapq.heappush(heap, (c1+1, k1))
        if -c2 - 1 > 0: heapq.heappush(heap, (c2+1, k2))
    if heap: res.append(heap[0][1])
    result = ''.join(res)
    return result if len(result) == len(s) else ''

Largest Number — LeetCode #179 Medium
Pattern: Custom comparator sort · O(n log n)

Given a list of non-negative integers, arrange them to form the largest number. Return as string.

Custom sort: compare strings a, b by whether a+b > b+a (as strings). Sort descending by this comparator. Handle the all-zeros edge case.
C# 
public string LargestNumber(int[] nums) {
    var strs = nums.Select(n => n.ToString()).ToArray();
    Array.Sort(strs, (string a, string b) =>
        string.Compare(b+a, a+b, StringComparison.Ordinal));
    if (strs[0] == "0") return "0";
    return string.Concat(strs);
}
Python 
from functools import cmp_to_key
def largestNumber(nums):
    strs = list(map(str, nums))
    strs.sort(key=cmp_to_key(lambda a, b: (1 if b+a > a+b else -1)))
    result = ''.join(strs)
    return '0' if result[0] == '0' else result

IPO — LeetCode #502 Hard
Pattern: Dual-heap greedy · O(n log n)

You start with capital w. At each of k steps, choose a project (profit[i], capital[i]) with capital[i] = current_w to maximize final capital.

Min-heap by capital: add all affordable projects to a max-heap by profit. Pick highest profit project. Repeat k times. When capital grows, more projects become affordable.
C# 
public int FindMaximizedCapital(int k, int w, int[] profits, int[] capital) {
    var capHeap = new SortedSet<(int c, int p, int i)>(
        Comparer<(int,int,int)>.Create((a,b) => a.c != b.c ? a.c-b.c : a.i-b.i));
    for (int i = 0; i < profits.Length; i++) capHeap.Add((capital[i], profits[i], i));
    var profHeap = new PriorityQueue<int, int>(Comparer<int>.Create((a,b) => b-a));
    while (k-- > 0) {
        while (capHeap.Count > 0 && capHeap.Min.c <= w) {
            var min = capHeap.Min; capHeap.Remove(min);
            profHeap.Enqueue(min.p, -min.p);
        }
        if (profHeap.Count == 0) break;
        w += profHeap.Dequeue();
    }
    return w;
}
Python 
import heapq
def findMaximizedCapital(k, w, profits, capital):
    cap_heap = sorted(zip(capital, profits))
    profit_heap = []
    i = 0
    for _ in range(k):
        while i < len(cap_heap) and cap_heap[i][0] <= w:
            heapq.heappush(profit_heap, -cap_heap[i][1]); i += 1
        if not profit_heap: break
        w -= heapq.heappop(profit_heap)
    return w
