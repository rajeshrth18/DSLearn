#1 Two Sum Easy LeetCode #1 · ?? Most common FAANG problem

Given an array nums and a target, return indices of the two numbers that add up to target. Exactly one solution exists. May not use the same element twice.

Time: O(n)Space: O(n)Pattern: Hash Map
Use a hash map to store value ? index. For each element, check if target - element exists in the map before inserting the element.
C# 
public int[] TwoSum(int[] nums, int target) {
    var map = new Dictionary<int,int>();
    for (int i = 0; i < nums.Length; i++) {
        int comp = target - nums[i];
        if (map.ContainsKey(comp)) return new[] {map[comp], i};
        map[nums[i]] = i;
    }
    return Array.Empty<int>();
}
Python 
def twoSum(nums, target):
    seen = {}
    for i, v in enumerate(nums):
        if target-v in seen: return [seen[target-v], i]
        seen[v] = i
#2 Best Time to Buy and Sell Stock Easy LeetCode #121

Given prices array, find the maximum profit from buying on one day and selling on a later day.

Time: O(n)Space: O(1)Pattern: Running Min
Track the minimum price seen so far. For each price, compute potential profit (price - minSoFar) and update the maximum profit.
C# 
public int MaxProfit(int[] prices) {
    int minP = int.MaxValue, best = 0;
    foreach (int p in prices) {
        minP = Math.Min(minP, p);
        best = Math.Max(best, p - minP);
    }
    return best;
}
Python

#4 Product of Array Except Self Medium LeetCode #238 · No division allowed

Return array where each element is the product of all other elements. Must run in O(n) without using division.

Time: O(n)Space: O(1) extraPattern: Prefix/Suffix Products
For each position i: result[i] = (product of all elements to the left) × (product of all elements to the right). Do two passes: left-to-right building prefix products, then right-to-left multiplying suffix products.
C# 
public int[] ProductExceptSelf(int[] nums) {
    int n = nums.Length; var res = new int[n];
    res[0] = 1;
    for (int i = 1; i < n; i++) res[i] = res[i-1] * nums[i-1];
    int right = 1;
    for (int i = n-1; i >= 0; i--) { res[i] *= right; right *= nums[i]; }
    return res;
}
Python 
def productExceptSelf(nums):
    n = len(nums); res = [1]*n
    for i in range(1, n): res[i] = res[i-1] * nums[i-1]
    r = 1
    for i in range(n-1, -1, -1): res[i] *= r; r *= nums[i]
    return res
#5 Maximum Subarray (Kadane's Algorithm) Medium LeetCode #53 · Classic

Find the contiguous subarray (at least one element) which has the largest sum.

Time: O(n)Space: O(1)Pattern: Kadane / DP
At each position, decide: extend the previous subarray or start a new one. curr = max(nums[i], curr + nums[i]). Track global max.
C# 
public int MaxSubArray(int[] nums) {
    int curr = nums[0], best = nums[0];
    for (int i = 1; i < nums.Length; i++) {
        curr = Math.Max(nums[i], curr + nums[i]);
        best = Math.Max(best, curr);
    }
    return best;
}
Python 
def maxSubArray(nums):
    curr = best = nums[0]
    for x in nums[1:]: curr = max(x, curr+x); best = max(best, curr)
    return best

#6 Valid Anagram Easy LeetCode #242

Given two strings s and t, return true if t is an anagram of s.

Time: O(n)Space: O(1) (26 chars)Pattern: Frequency Count
Count character frequencies. Two strings are anagrams iff they have identical frequency tables. Either use two frequency arrays, or use one and increment for s, decrement for t, check all zeros.
C# 
public bool IsAnagram(string s, string t) {
    if (s.Length != t.Length) return false;
    var cnt = new int[26];
    foreach (char c in s) cnt[c-'a']++;
    foreach (char c in t) if (--cnt[c-'a'] < 0) return false;
    return true;
}
Python 
from collections import Counter
def isAnagram(s, t): return Counter(s) == Counter(t)
#7 Group Anagrams Medium LeetCode #49

Given an array of strings, group the anagrams together.

Time: O(n·k log k)Space: O(n·k)Pattern: HashMap with sorted key
Anagrams have the same sorted form. Use a dictionary keyed by the sorted string. Group all strings with the same sorted form together.
C# 
public IList<IList<string>> GroupAnagrams(string[] strs) {
    var map = new Dictionary<string, List<string>>();
    foreach (var s in strs) {
        var key = new string(s.OrderBy(c => c).ToArray());
        if (!map.ContainsKey(key)) map[key] = new List<string>();
        map[key].Add(s);
    }
    return map.Values.ToList<IList<string>>();
}

#8 Valid Parentheses Easy LeetCode #20

Determine if a string of brackets '(', ')', '{', '}', '[', ']' is valid (properly opened and closed).

Time: O(n)Space: O(n)Pattern: Stack
Use a stack. Push opening brackets. When you see a closing bracket, check if the stack top is the matching opener. If not, or stack is empty, return false. At end, stack should be empty.
C# 
public bool IsValid(string s) {
    var st = new Stack<char>();
    foreach (char c in s) {
        if ("([{".Contains(c)) st.Push(c);
        else {
            if (st.Count == 0) return false;
            char top = st.Pop();
            if (c==')' && top!='(' || c==']' && top!='[' || c=='}' && top!='{') return false;
        }
    }
    return st.Count == 0;
}
Python 
def isValid(s):
    st = []; m = {')':'(', ']':'[', '}':'{'}
    for c in s:
        if c not in m: st.append(c)
        elif not st or st[-1] != m[c]: return False
        else: st.pop()
    return not st
#9 Longest Consecutive Sequence Medium LeetCode #128

Given an unsorted array, find the length of the longest consecutive elements sequence. Must run in O(n).

Time: O(n)Space: O(n)Pattern: HashSet, start detection
Put all numbers in a HashSet. For each number n, only start counting if n-1 is NOT in the set (it's the start of a sequence). Then count n, n+1, n+2... This ensures each sequence is counted exactly once.
C# 
public int LongestConsecutive(int[] nums) {
    var set = new HashSet<int>(nums);
    int best = 0;
    foreach (int n in set) {
        if (!set.Contains(n - 1)) {
            int len = 1;
            while (set.Contains(n + len)) len++;
            best = Math.Max(best, len);
        }
    }
    return best;
}
Python 
def longestConsecutive(nums):
    s = set(nums); best = 0
    for n in s:
        if n-1 not in s:
            length = 1
            while n+length in s: length += 1
            best = max(best, length)
    return best
#10 Trapping Rain Water Hard LeetCode #42 · Classic

Given heights array, compute how much water can be trapped after rain.

Time: O(n)Space: O(1)Pattern: Two Pointers
Two-pointer approach: maintain maxLeft and maxRight. Water at position i = min(maxLeft, maxRight) - height[i]. Process the side with the smaller max pointer first — that side's water is determined.
C# 
public int Trap(int[] h) {
    int L=0, R=h.Length-1, maxL=0, maxR=0, water=0;
    while (L < R) {
        if (h[L] < h[R]) {
            maxL = Math.Max(maxL, h[L]);
            water += maxL - h[L++];
        } else {
            maxR = Math.Max(maxR, h[R]);
            water += maxR - h[R--];
        }
    }
    return water;
}
Python 
def trap(height):
    L, R, maxL, maxR, w = 0, len(height)-1, 0, 0, 0
    while L < R:
        if height[L] < height[R]:
            maxL = max(maxL, height[L]); w += maxL-height[L]; L+=1
        else:
            maxR = max(maxR, height[R]); w += maxR-height[R]; R-=1
    return w

#11 3Sum Medium LeetCode #15

Find all unique triplets that sum to zero.

Time: O(n˛)Space: O(1)Pattern: Sort + Two Pointers
Sort the array. Fix one element i, then use two pointers L and R on the rest to find pairs summing to -nums[i]. Skip duplicates at each level to avoid duplicate triplets.
C# 
public IList<IList<int>> ThreeSum(int[] nums) {
    Array.Sort(nums); var res = new List<IList<int>>();
    for (int i=0; i<nums.Length-2; i++) {
        if (i>0 && nums[i]==nums[i-1]) continue;
        int L=i+1, R=nums.Length-1;
        while (L<R) {
            int s=nums[i]+nums[L]+nums[R];
            if(s==0){
                res.Add(new[]{nums[i],nums[L],nums[R]});
                while(L<R&&nums[L]==nums[L+1])L++;
                while(L<R&&nums[R]==nums[R-1])R--;
                L++; R--;
            } else if(s<0) L++; else R--;
        }
    }
    return res;
}
Python 
def threeSum(nums):
    nums.sort(); res = []
    for i in range(len(nums)-2):
        if i > 0 and nums[i]==nums[i-1]: continue
        L, R = i+1, len(nums)-1
        while L < R:
            s = nums[i]+nums[L]+nums[R]
            if s == 0:
                res.append([nums[i],nums[L],nums[R]])
                while L < R and nums[L]==nums[L+1]: L+=1
                while L < R and nums[R]==nums[R-1]: R-=1
                L+=1; R-=1
            elif s < 0: L+=1
            else: R-=1
    return res

#13 Find Duplicate Number Medium LeetCode #287 · Floyd's cycle

Given n+1 integers in [1,n], find the one duplicate. Must not modify the array, O(1) extra space.

Time: O(n)Space: O(1)Pattern: Floyd's Cycle Detection
Treat the array as a linked list where index i points to nums[i]. There's a cycle iff there's a duplicate. Find the cycle start using Floyd's algorithm (fast/slow pointers). The cycle entry = duplicate number.
C# 
public int FindDuplicate(int[] nums) {
    int slow=nums[0], fast=nums[0];
    do { slow=nums[slow]; fast=nums[nums[fast]]; } while (slow!=fast);
    slow=nums[0];
    while(slow!=fast){ slow=nums[slow]; fast=nums[fast]; }
    return slow;
}
Python 
def findDuplicate(nums):
    slow = fast = nums[0]
    while True:
        slow = nums[slow]; fast = nums[nums[fast]]
        if slow == fast: break
    slow = nums[0]
    while slow != fast: slow = nums[slow]; fast = nums[fast]
    return slow
#14 Encode and Decode Strings Medium LeetCode #271

Design an algorithm to encode a list of strings to a single string and decode it back. No special character is forbidden in input.

Time: O(n)Space: O(n)Pattern: Length-prefix encoding
Prefix each string with its length and a delimiter: "4#word". Decoding: read the number before '#' to know how many characters to read next. This handles any characters in the string (even '#').
C# 
public string Encode(IList<string> strs)
    => string.Concat(strs.Select(s => $"{s.Length}#{s}"));

public IList<string> Decode(string s) {
    var res=new List<string>(); int i=0;
    while(i<s.Length){
        int j=s.IndexOf('#',i); int n=int.Parse(s[i..j]);
        res.Add(s.Substring(j+1,n)); i=j+1+n;
    }
    return res;
}
Python 
def encode(strs): return ''.join(f'{len(s)}#{s}' for s in strs)

def decode(s):
    res, i = [], 0
    while i < len(s):
        j = s.index('#', i); n = int(s[i:j])
        res.append(s[j+1:j+1+n]); i = j+1+n
    return res
#15 Minimum Window Substring Hard LeetCode #76

Find the minimum window in s that contains all characters of t.

Time: O(n+m)Space: O(m)Pattern: Variable Sliding Window
Expand right until all t characters are covered. Then shrink left while maintaining coverage. Track minimum window. Use a "formed" counter to know when all required characters have sufficient frequency.
C# 
public string MinWindow(string s, string t) {
    var need=new Dictionary<char,int>();
    foreach(char c in t) need[c]=need.GetValueOrDefault(c)+1;
    int have=0,req=need.Count,L=0,mLen=int.MaxValue,mL=0;
    var win=new Dictionary<char,int>();
    for(int R=0;R<s.Length;R++){
        char c=s[R]; win[c]=win.GetValueOrDefault(c)+1;
        if(need.ContainsKey(c)&&win[c]==need[c]) have++;
        while(have==req){
            if(R-L+1<mLen){mLen=R-L+1;mL=L;}
            char lc=s[L++]; win[lc]--;
            if(need.ContainsKey(lc)&&win[lc]<need[lc]) have--;
        }
    }
    return mLen==int.MaxValue?"":s.Substring(mL,mLen);
}
