Section 1 — Palindrome Review

A palindrome reads the same forwards and backwards. Finding the longest palindromic substring is a classic interview problem.

  • Odd-length palindromes: centered at a single character (e.g., "racecar" centered at 'e')
  • Even-length palindromes: centered between two characters (e.g., "abba" centered between 'b' and 'b')

1.1 — Expand Around Center: O(n˛) Approach

For each center (n odd-centers + n-1 even-centers), expand outward while characters match. Simple but O(n˛).

C#
static string LongestPalindromeExpand(string s)
{
    int n = s.Length, start = 0, maxLen = 1;

    void Expand(int l, int r)
    {
        while (l >= 0 && r < n && s[l] == s[r]) { l--; r++; }
        if (r - l - 1 > maxLen) { maxLen = r - l - 1; start = l + 1; }
    }

    for (int i = 0; i < n; i++)
    {
        Expand(i, i);     // odd-length
        Expand(i, i + 1); // even-length
    }
    return s.Substring(start, maxLen);
}
Section 2 — Manacher's Algorithm: O(n)

Manacher's algorithm preprocesses the string by inserting a separator character # between every character (and at both ends). This transforms the problem: all palindromes become odd-length in the new string, eliminating the even/odd case split.

Transform "abba" ? "#a#b#b#a#" Index: 0 1 2 3 4 5 6 7 8 P[i] = radius of palindrome centered at i (in transformed string) P = [0, 1, 0, 1, 4, 1, 0, 1, 0] ? ? "a" "abba" — P[4]=4 means 4 chars on each side match Extract original: center=4, P[4]=4 ? original palindrome starts at (4-4)/2 = 0, length P[4]=4 ? "abba"
?? The Mirror Trick: Manacher maintains a "rightmost palindrome" boundary [c, r]. For position i inside this boundary, we know P[i] = P[mirror(i)] where mirror = 2*c - i. This avoids re-expanding palindromes we've already computed.
C#
static string LongestPalindromeManacher(string s)
{
    // Transform: "abc" ? "#a#b#c#"
    var t = new System.Text.StringBuilder("#");
    foreach (char c in s) { t.Append(c); t.Append('#'); }
    string T = t.ToString();
    int n = T.Length;
    int[] P = new int[n];
    int c = 0, r = 0;  // center and right boundary of rightmost palindrome

    for (int i = 0; i < n; i++)
    {
        int mirror = 2 * c - i;
        if (i < r)
            P[i] = Math.Min(r - i, P[mirror]);

        // Try to expand beyond known boundary
        while (i - P[i] - 1 >= 0 && i + P[i] + 1 < n && T[i - P[i] - 1] == T[i + P[i] + 1])
            P[i]++;

        if (i + P[i] > r)
        {
            c = i;
            r = i + P[i];
        }
    }

    // Find maximum P[i]
    int maxLen = 0, center = 0;
    for (int i = 0; i < n; i++)
        if (P[i] > maxLen) { maxLen = P[i]; center = i; }

    int start = (center - maxLen) / 2;
    return s.Substring(start, maxLen);
}
// "babad" ? "bab" or "aba"   "cbbd" ? "bb"
Python
def longest_palindrome_manacher(s: str) -> str:
    T = '#' + '#'.join(s) + '#'
    n = len(T)
    P = [0] * n
    c = r = 0

    for i in range(n):
        mirror = 2 * c - i
        if i < r:
            P[i] = min(r - i, P[mirror])
        while (i - P[i] - 1 >= 0 and i + P[i] + 1 < n
               and T[i - P[i] - 1] == T[i + P[i] + 1]):
            P[i] += 1
        if i + P[i] > r:
            c, r = i, i + P[i]

    max_len = max(P)
    center = P.index(max_len)
    start = (center - max_len) // 2
    return s[start : start + max_len]

print(longest_palindrome_manacher("babad"))   # "bab"
print(longest_palindrome_manacher("cbbd"))    # "bb"
Section 3 — Count All Palindromic Substrings

Using the P array, we can also count all palindromic substrings in O(n). Each P[i] contributes ?P[i]/2? + (1 if T[i]!='#') palindromes... but the simplest way: each position i contributes (P[i] + 1) / 2 palindromes from the original string.

Python
def count_palindromic_substrings(s: str) -> int:
    T = '#' + '#'.join(s) + '#'
    n = len(T)
    P = [0] * n
    c = r = 0

    for i in range(n):
        mirror = 2 * c - i
        if i < r:
            P[i] = min(r - i, P[mirror])
        while (i - P[i] - 1 >= 0 and i + P[i] + 1 < n
               and T[i - P[i] - 1] == T[i + P[i] + 1]):
            P[i] += 1
        if i + P[i] > r:
            c, r = i, i + P[i]

    # Each P[i] counts palindromes: (P[i]+1)//2 palindromes contributed
    return sum((p + 1) // 2 for p in P)

print(count_palindromic_substrings("abc"))   # 3
print(count_palindromic_substrings("aaa"))   # 6
?? Key Takeaways:
  • Transform with # separators unifies odd/even palindromes — all become odd-length
  • Manacher's maintains [c, r] — the rightmost palindrome — to reuse computed radii
  • Time: O(n); Space: O(n) for the P array and transformed string
  • Expand-around-center is simpler but O(n˛); use Manacher for competitive programming
