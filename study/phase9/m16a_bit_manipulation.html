Section 1 — Binary Representation

Every integer is stored in binary (base-2). Understanding how bits work is essential for writing efficient code and solving classic interview problems in O(1) or O(log n).

Integer 42 in binary (32-bit int): Decimal: 42 Binary: 0000 0000 0000 0000 0000 0000 0010 1010 ↑ ↑↑ bit5=1 bit3=1, bit1=1 2^5 + 2^3 + 2^1 = 32+8+2 = 42 ✓ Bit positions: rightmost = bit 0 (least significant) = LSB leftmost = bit 31 (most significant) = MSB Negative numbers (two's complement): -1 in 32 bits = 1111 1111 1111 1111 1111 1111 1111 1111 -42 = flip bits of 42, then add 1

Section 3 — Essential Bit Tricks
C#
// ── Single Bit Operations ────────────────────────────────
bool GetBit(int n, int i)  => (n & (1 << i)) != 0;   // is bit i set?
int  SetBit(int n, int i)  => n | (1 << i);            // set bit i to 1
int  ClearBit(int n, int i) => n & ~(1 << i);         // clear bit i to 0
int  ToggleBit(int n, int i) => n ^ (1 << i);          // flip bit i

// ── Check/Test ───────────────────────────────────────────
bool IsEven(int n) => (n & 1) == 0;          // last bit = 0 → even
bool IsPowerOf2(int n) => n > 0 && (n & (n-1)) == 0;  // only one bit set

// ── Remove / Isolate ─────────────────────────────────────
int RemoveLSB(int n) => n & (n - 1);   // clear lowest set bit  (e.g. 12→8)
int IsolateLSB(int n) => n & (-n);      // keep only lowest set bit (e.g. 12→4)
int IsolateMSB(int n) => 1 << (31 - System.Numerics.BitOperations.LeadingZeroCount((uint)n));

// ── Absolute Value without branch ───────────────────────
int AbsNoBranch(int n)
{
    int mask = n >> 31;  // all 0s if positive, all 1s if negative
    return (n ^ mask) - mask;
}

// ── Swap without temp ────────────────────────────────────
void SwapXOR(ref int a, ref int b)
{
    if (a == b) return;  // IMPORTANT: same variable check
    a ^= b;  b ^= a;  a ^= b;
}

// ── Count set bits (popcount) ────────────────────────────
int Popcount(int n)
{
    int count = 0;
    while (n != 0) { n &= (n - 1); count++; }  // Brian Kernighan's
    return count;
}
// Or: System.Numerics.BitOperations.PopCount((uint)n)
Python
# ── Single Bit Operations ────────────────────────────────
get_bit    = lambda n, i: (n >> i) & 1           # is bit i set?
set_bit    = lambda n, i: n | (1 << i)            # set bit i to 1

# ── Check/Test ───────────────────────────────────────────
is_even        = lambda n: (n & 1) == 0

# ── Remove / Isolate ─────────────────────────────────────
remove_lsb  = lambda n: n & (n - 1)   # clear lowest set bit
isolate_lsb = lambda n: n & (-n)        # keep only lowest set bit

# ── Count set bits ───────────────────────────────────────
popcount_slow = lambda n: bin(n).count('1')   # Python's shortcut

def popcount(n: int) -> int:             # Brian Kernighan's
    count = 0
    while n:
        n &= n - 1
        count += 1
    return count

# Examples
print(is_power_of_2(16))   # True
print(is_power_of_2(10))   # False
print(popcount(42))        # 3 (42 = 101010)
print(remove_lsb(12))     # 8  (12=1100 → 1000)
print(isolate_lsb(12))    # 4  (12=1100 → 0100)
Section 4 — Bitmask Techniques

4.1 — Enumerate All Subsets of a Set

A bitmask of n bits can represent any subset of an n-element set. Iterate from 0 to 2^n - 1 to enumerate all 2^n subsets in O(2^n).

C#
static void AllSubsets(int[] arr)
{
    int n = arr.Length;
    for (int mask = 0; mask < (1 << n); mask++)
    {
        Console.Write("{ ");
        for (int i = 0; i < n; i++)
            if ((mask & (1 << i)) != 0)
                Console.Write(arr[i] + " ");
        Console.WriteLine("}");
    }
}
AllSubsets(new[] { 1, 2, 3 });
// {} {1} {2} {1,2} {3} {1,3} {2,3} {1,2,3}

// Enumerate all proper subsets of a mask (DP on subsets trick)
static void EnumerateSubsetsOf(int mask)
{
    for (int sub = mask; sub > 0; sub = (sub - 1) & mask)
        Console.Write(sub + " ");
}// mask=7(111): 7,6,5,4,3,2,1
Python
def all_subsets(arr: list) -> list:
    n = len(arr)
    result = []
    for mask in range(1 << n):
        subset = [arr[i] for i in range(n) if mask >> i & 1]
        result.append(subset)
    return result

print(all_subsets([1, 2, 3]))
# [[], [1], [2], [1,2], [3], [1,3], [2,3], [1,2,3]]

# Enumerate proper subsets of a mask
def subsets_of(mask: int):
    sub = mask
    while sub:
        yield sub
        sub = (sub - 1) & mask
📌 Bitmask DP: Many DP problems on small sets (n ≤ 20) use bitmask DP. State = a bitmask representing which elements have been used. Classic: Travelling Salesman, Minimum Cost to Connect All Nodes, Assignment Problems.
