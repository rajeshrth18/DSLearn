Problem 1 — Single Number (LC 136)
Single Number
LeetCode 136  |  Easy  |  XOR trick

Given a non-empty array where every element appears twice except one, find the one that appears only once. Must run in O(n) time and O(1) space.

Key insight: XOR of a number with itself = 0. XOR with 0 = the number. So XOR all elements — pairs cancel out, leaving the unique one.

4 ^ 1 ^ 2 ^ 1 ^ 2 = 4

C#
public int SingleNumber(int[] nums)
{
    int result = 0;
    foreach (var n in nums) result ^= n;
    return result;   // O(n) time, O(1) space
}
Python
from functools import reduce
from operator  import xor

def single_number(nums: list[int]) -> int:
    return reduce(xor, nums)   # or: result=0; for n in nums: result^=n

print(single_number([4, 1, 2, 1, 2]))  # 4
Problem 2 — Single Number II (LC 137)
Single Number II
LeetCode 137  |  Medium  |  Bit counting

Every element appears exactly three times except one. Find that unique element in O(n) time and O(1) space.

Key insight: For each bit position, count how many numbers have that bit set. If count % 3 != 0, the unique number has that bit set.

C#
public int SingleNumber2(int[] nums)
{
    int result = 0;
    for (int bit = 0; bit < 32; bit++)
    {
        int sum = 0;
        foreach (var n in nums) sum += (n >> bit) & 1;
        if (sum % 3 != 0) result |= 1 << bit;
    }
    return result;
}
Python
def single_number2(nums: list[int]) -> int:
    result = 0
    for bit in range(32):
        total = sum((n >> bit) & 1 for n in nums)
        if total % 3:
            result |= 1 << bit
    # Handle negative numbers in Python (no 32-bit overflow)
    if result >= (1 << 31):
        result -= (1 << 32)
    return result
Problem 3 — Counting Bits (LC 338)
Counting Bits
LeetCode 338  |  Easy  |  DP + Bit

Given n, return an array of length n+1 where answer[i] = number of 1-bits in binary representation of i.

Key insight: dp[i] = dp[i >> 1] + (i & 1) — right-shift removes one bit; the LSB tells you if that bit was 1.

C#
public int[] CountBits(int n)
{
    int[] dp = new int[n + 1];
    for (int i = 1; i <= n; i++)
        dp[i] = dp[i >> 1] + (i & 1);
    return dp;   // O(n) time and space
}
Python
def count_bits(n: int) -> list[int]:
    dp = [0] * (n + 1)
    for i in range(1, n + 1):
        dp[i] = dp[i >> 1] + (i & 1)
    return dp

print(count_bits(5))  # [0, 1, 1, 2, 1, 2]
Problem 4 — Missing Number (LC 268)
Missing Number
LeetCode 268  |  Easy  |  XOR or Math

Given array [0..n] with one number missing, find the missing number.

XOR approach: XOR indices 0..n with all array values — everything cancels except the missing number.

Math approach: Expected sum = n*(n+1)/2. Missing = expected - actual sum.

C#
public int MissingNumber(int[] nums)
{
    int missing = nums.Length;  // start with XOR of n
    for (int i = 0; i < nums.Length; i++)
        missing ^= i ^ nums[i];
    return missing;
}

// Alternative: Math
public int MissingNumberMath(int[] nums)
{
    int n = nums.Length;
    return n * (n + 1) / 2 - nums.Sum();
}
Python
def missing_number(nums: list[int]) -> int:
    n = len(nums)
    return n * (n + 1) // 2 - sum(nums)

print(missing_number([3, 0, 1]))  # 2
Problem 5 — Find Two Non-Repeating Numbers (LC 260)
Single Number III — Two Unique Elements
LeetCode 260  |  Medium  |  XOR partition

Every element appears twice except two distinct numbers. Find both in O(n) time and O(1) space.

Approach: XOR all ? get xor = a^b. Find any set bit in xor (use xor & -xor to isolate LSB). Partition numbers by that bit ? XOR each partition to get a and b.

C#
public int[] SingleNumber3(int[] nums)
{
    int xorAll = 0;
    foreach (var n in nums) xorAll ^= n;  // xorAll = a ^ b

    int diff = xorAll & (-xorAll);         // lowest set bit
    int a = 0, b = 0;
    foreach (var n in nums)
    {
        if ((n & diff) == 0) a ^= n;  // group 0: bit not set
        else                   b ^= n;  // group 1: bit is set
    }
    return new[] { a, b };
}
Python
def single_number3(nums: list[int]) -> list[int]:
    xor_all = 0
    for n in nums: xor_all ^= n       # xor_all = a ^ b
    diff = xor_all & (-xor_all)          # isolate lowest set bit
    a = b = 0
    for n in nums:
        if n & diff: b ^= n
        else:        a ^= n
    return [a, b]

print(single_number3([1, 2, 1, 3, 2, 5]))  # [3, 5]
Problem 6 — Reverse Bits (LC 190)
C#
public uint ReverseBits(uint n)
{
    uint result = 0;
    for (int i = 0; i < 32; i++)
    {
        result = (result << 1) | (n & 1);
        n >>= 1;
    }
    return result;
}
Python
def reverse_bits(n: int) -> int:
    result = 0
    for _ in range(32):
        result = (result << 1) | (n & 1)
        n >>= 1
    return result

print(reverse_bits(0b00000010100101000001111010011100))
