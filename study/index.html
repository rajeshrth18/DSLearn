Data Structure Access Search Insert Delete Space

StackO(1)O(n)O(1)O(1)O(n)
QueueO(1)O(n)O(1)O(1)O(n)
Binary Search TreeO(log n)O(log n)O(log n)O(log n)O(n)
AVL / Red-Black TreeO(log n)O(log n)O(log n)O(log n)O(n)
Hash Table (Dictionary)O(1)*O(1)*O(1)*O(1)*O(n)
Heap (Min/Max)O(1) topO(n)O(log n)O(log n)O(n)
TrieO(k)O(k)O(k)O(k)O(n·k)
Segment TreeO(log n)O(log n)O(log n)O(log n)O(n)

* Amortized · ** With reference to node · k = key/string length · log n = balanced tree height
