💡 How to Approach Tree Problems:
1. Draw the tree for the given example
2. Ask: Do I need DFS or BFS? Do I process top-down or bottom-up?
3. Define what your recursive function returns — make it clear
4. Handle the base case (null node) first
Easy Problems (1–5)
#1Invert Binary Tree
EasyDFS

Invert a binary tree (mirror it). Return its root.

Input: Output: [4] [4] / \ / \ [2] [7] [7] [2] / \ / \ / \ / \ [1][3][6][9] [9][6] [3][1]
Approach: Post-order DFS: invert left subtree, invert right subtree, then swap left and right children of current node.
public TreeNode? InvertTree(TreeNode? root)
{
    if (root == null) return null;
    (root.Left, root.Right) = (InvertTree(root.Right), InvertTree(root.Left));
    return root;
}
// Time: O(n), Space: O(h)
#2Symmetric Tree
EasyDFS

Check whether a binary tree is a mirror of itself (symmetric around its center).

Example: [1,2,2,3,4,4,3] → true. [1,2,2,null,3,null,3] → false
Approach: Compare the left subtree with a mirrored right subtree. isMirror(left, right): check left.val == right.val AND isMirror(left.left, right.right) AND isMirror(left.right, right.left).
public bool IsSymmetric(TreeNode? root)
    => IsMirror(root?.Left, root?.Right);

private bool IsMirror(TreeNode? l, TreeNode? r)
{
    if (l == null && r == null) return true;
    if (l == null || r == null) return false;
    return l.Val == r.Val
        && IsMirror(l.Left,  r.Right)
        && IsMirror(l.Right, r.Left);
}
// Time: O(n), Space: O(h)
#3Maximum Depth of Binary Tree
EasyDFS or BFS

Given the root of a binary tree, return its maximum depth (number of nodes along the longest root-to-leaf path).

Approach: Simple recursive post-order: return 1 + max(depth(left), depth(right)). Base case: null = 0.
public int MaxDepth(TreeNode? root)
{
    if (root == null) return 0;
    return 1 + Math.Max(MaxDepth(root.Left), MaxDepth(root.Right));
}
// Time: O(n), Space: O(h)
#4Same Tree
EasyDFS

Given the roots of two binary trees, check if they are the same (structurally identical and nodes have the same values).

Approach: Recursively check: both null → true; one null → false; values different → false; recurse on both sides.
public bool IsSameTree(TreeNode? p, TreeNode? q)
{
    if (p == null && q == null) return true;
    if (p == null || q == null) return false;
    return p.Val == q.Val
        && IsSameTree(p.Left,  q.Left)
        && IsSameTree(p.Right, q.Right);
}
// Time: O(n), Space: O(h)
#5Path Sum I
EasyDFS

Given a binary tree and target sum, return true if any root-to-leaf path sums to target.

Approach: Subtract current node value from target. At a leaf, check if remaining is 0.
public bool HasPathSum(TreeNode? node, int target)
{
    if (node == null) return false;
    target -= node.Val;
    if (node.Left == null && node.Right == null)
        return target == 0;
    return HasPathSum(node.Left, target) || HasPathSum(node.Right, target);
}
// Time: O(n), Space: O(h)
Medium Problems (6–12)
#6Binary Tree Level Order Traversal
MediumBFSQueue

Return level-order traversal of a tree's node values as a list of lists (each sublist = one level).

Approach: BFS with queue. Snapshot queue.Count at start of each level — that's how many nodes belong to this level.
public IList<IList<int>> LevelOrder(TreeNode? root)
{
    var result = new List<IList<int>>();
    if (root == null) return result;
    var q = new Queue<TreeNode>();
    q.Enqueue(root);
    while (q.Count > 0)
    {
        int sz = q.Count;
        var level = new List<int>();
        for (int i = 0; i < sz; i++)
        {
            var n = q.Dequeue();
            level.Add(n.Val);
            if (n.Left  != null) q.Enqueue(n.Left);
            if (n.Right != null) q.Enqueue(n.Right);
        }
        result.Add(level);
    }
    return result;
}
// Time: O(n), Space: O(n)
#7Kth Smallest Element in BST
MediumIn-OrderBST

Given the root of a BST and integer k, return the kth smallest value (1-indexed).

Approach: In-order traversal of a BST produces sorted values. Count down k during in-order — stop when k reaches 0.
public int KthSmallest(TreeNode? root, int k)
{
    int result = 0;
    InOrder(root, ref k, ref result);
    return result;
}
private void InOrder(TreeNode? n, ref int k, ref int res)
{
    if (n == null || k == 0) return;
    InOrder(n.Left, ref k, ref res);
    if (--k == 0) { res = n.Val; return; }
    InOrder(n.Right, ref k, ref res);
}
// Time: O(h + k), Space: O(h)
#8Validate Binary Search Tree
MediumBST Validation

Given the root of a binary tree, determine if it is a valid BST.

Approach: Pass min/max bounds down. Each node must be strictly in (min, max). Left subtree: max becomes node.val. Right subtree: min becomes node.val.
public bool IsValidBST(TreeNode? root)
    => Validate(root, long.MinValue, long.MaxValue);

private bool Validate(TreeNode? n, long min, long max)
{
    if (n == null) return true;
    if (n.Val <= min || n.Val >= max) return false;
    return Validate(n.Left,  min,    n.Val) &&
           Validate(n.Right, n.Val, max);
}
// Time: O(n), Space: O(h)
#9Construct Binary Tree from Preorder and Inorder
MediumDivide & Conquer

Given preorder and inorder traversal arrays, reconstruct the binary tree.

Example: preorder=[3,9,20,15,7], inorder=[9,3,15,20,7]
preorder: [3, 9, 20, 15, 7] — first element is always root inorder: [9, 3, 15, 20, 7] — root splits left and right subtrees Root = 3. In inorder: index of 3 = 1. Left subtree: inorder=[9], preorder=[9] Right subtree: inorder=[15,20,7], preorder=[20,15,7]
Approach: preorder[0] = root. Find root in inorder → left subtree has that many elements. Use HashMap for O(1) inorder lookup.
public TreeNode? BuildTree(int[] preorder, int[] inorder)
{
    var indexMap = new Dictionary<int, int>();
    for (int i = 0; i < inorder.Length; i++) indexMap[inorder[i]] = i;
    int preIdx = 0;
    return Build(preorder, 0, inorder.Length - 1, indexMap, ref preIdx);
}
private TreeNode? Build(int[] pre, int inL, int inR,
    Dictionary<int, int> map, ref int preIdx)
{
    if (inL > inR) return null;
    int rootVal = pre[preIdx++];
    var node = new TreeNode(rootVal);
    int mid = map[rootVal];
    node.Left  = Build(pre, inL,    mid - 1, map, ref preIdx);
    node.Right = Build(pre, mid + 1, inR,     map, ref preIdx);
    return node;
}
// Time: O(n), Space: O(n)
#10Binary Tree Right Side View
MediumBFS

Given the root of a binary tree, imagine you're standing on the right side. Return the values of the nodes visible (rightmost node of each level).

Approach: BFS level-order. The last node dequeued on each level is visible from the right side.
public IList<int> RightSideView(TreeNode? root)
{
    var result = new List<int>();
    if (root == null) return result;
    var q = new Queue<TreeNode>();
    q.Enqueue(root);
    while (q.Count > 0)
    {
        int sz = q.Count;
        for (int i = 0; i < sz; i++)
        {
            var n = q.Dequeue();
            if (i == sz - 1) result.Add(n.Val);
            if (n.Left  != null) q.Enqueue(n.Left);
            if (n.Right != null) q.Enqueue(n.Right);
        }
    }
    return result;
}
// Time: O(n), Space: O(n)
#11Lowest Common Ancestor of BST
MediumBSTLCA

Given a BST and two nodes p, q, find their lowest common ancestor.

Approach: Use BST property. If both p and q are less than current → LCA in left. If both greater → LCA in right. Otherwise → current node is LCA.
public TreeNode? LowestCommonAncestor(TreeNode? root,
    TreeNode p, TreeNode q)
{
    while (root != null)
    {
        if (p.Val < root.Val && q.Val < root.Val)
            root = root.Left;
        else if (p.Val > root.Val && q.Val > root.Val)
            root = root.Right;
        else
            return root; // split point = LCA
    }
    return null;
}
// Time: O(h), Space: O(1)
#12Path Sum II — All Paths
MediumBacktrackingDFS

Find all root-to-leaf paths in a binary tree that sum to a given target. Return all such paths.

Approach: Backtracking DFS: add current node → recurse → remove (backtrack). At a leaf, if sum matches, copy path to results.
public IList<IList<int>> PathSum(TreeNode? root, int target)
{
    var result = new List<IList<int>>();
    DFS(root, target, new List<int>(), result);
    return result;
}
private void DFS(TreeNode? n, int rem, List<int> path,
                  List<IList<int>> result)
{
    if (n == null) return;
    path.Add(n.Val); rem -= n.Val;
    if (n.Left == null && n.Right == null && rem == 0)
        result.Add(new List<int>(path));
    DFS(n.Left, rem, path, result);
    DFS(n.Right, rem, path, result);
    path.RemoveAt(path.Count - 1); // backtrack
}
// Time: O(n²) worst case — O(n·h) more precisely
// Remember: new List<int>(path) copies the list — without this you store a reference!
Hard Problems (13–15)
#13Binary Tree Maximum Path Sum
HardDFSGlobal Max

Find the path in a binary tree with maximum sum. The path can start and end at any node.

Approach: Post-order. At each node: update global max with left_gain + node + right_gain. Return only the better single branch (node + max(left, right)) since a path can't branch when going up.
public int MaxPathSum(TreeNode? root)
{
    int max = int.MinValue;
    Gain(root, ref max);
    return max;
}
private int Gain(TreeNode? n, ref int max)
{
    if (n == null) return 0;
    int L = Math.Max(0, Gain(n.Left,  ref max));
    int R = Math.Max(0, Gain(n.Right, ref max));
    max = Math.Max(max, n.Val + L + R);
    return n.Val + Math.Max(L, R);
}
// [-10,9,20,15,7] → 42
// Time: O(n), Space: O(h)
#14Serialize and Deserialize Binary Tree
HardDesignPre-order

Design an algorithm to serialize a binary tree to a string and deserialize that string back to the original tree structure.

Approach: Pre-order traversal with "#" for null nodes. Serialize: pre-order append. Deserialize: queue of tokens, pre-order reconstruction.
public string Serialize(TreeNode? root)
{
    var sb = new System.Text.StringBuilder();
    Ser(root, sb);
    return sb.ToString();
}
private void Ser(TreeNode? n, System.Text.StringBuilder sb)
{
    if (n == null) { sb.Append("#,"); return; }
    sb.Append(n.Val).Append(',');
    Ser(n.Left, sb); Ser(n.Right, sb);
}
public TreeNode? Deserialize(string data)
{
    var tokens = new Queue<string>(data.Split(',',
        StringSplitOptions.RemoveEmptyEntries));
    return Des(tokens);
}
private TreeNode? Des(Queue<string> t)
{
    string v = t.Dequeue();
    if (v == "#") return null;
    var n = new TreeNode(int.Parse(v));
    n.Left = Des(t); n.Right = Des(t);
    return n;
}
// Time: O(n), Space: O(n)
#15Diameter of Binary Tree
HardDFSGlobal Max

Given the root of a binary tree, return the length of the diameter (longest path between any two nodes, measured in edges).

Approach: At each node, the diameter through it = left_height + right_height + 2. Track the max globally. Return height upward.
public int DiameterOfBinaryTree(TreeNode? root)
{
    int max = 0;
    Height(root, ref max);
    return max;
}
private int Height(TreeNode? n, ref int max)
{
    if (n == null) return -1;
    int L = Height(n.Left,  ref max);
    int R = Height(n.Right, ref max);
    max = Math.Max(max, L + R + 2);
    return 1 + Math.Max(L, R);
}
// [1,2,3,4,5] → 3
// Single node [1] → 0 (no edges)
// Time: O(n), Space: O(h)
