1. What Is Backtracking?

Backtracking is a systematic way to search through all possible solutions to a problem. It builds solutions incrementally, and when a partial solution can never lead to a complete valid solution, it backtracks — undoes the last choice and tries the next option.

Think of it as exploring a decision tree: at each node you make a choice, recurse deeper, and when you hit a dead end (or find a solution), you undo that choice and try another branch.

Backtracking decision tree for permutations of [1,2,3]: Start: [] ├── Choose 1: [1] │ ├── Choose 2: [1,2] │ │ └── Choose 3: [1,2,3] ✓ (add to results) │ └── Choose 3: [1,3] │ └── Choose 2: [1,3,2] ✓ ├── Choose 2: [2] │ ├── Choose 1: [2,1] │ │ └── Choose 3: [2,1,3] ✓ │ └── Choose 3: [2,3] │ └── Choose 1: [2,3,1] ✓ └── Choose 3: [3] ├── Choose 1: [3,1] │ └── Choose 2: [3,1,2] ✓ └── Choose 2: [3,2] └── Choose 1: [3,2,1] ✓ Each leaf is a solution. Backtrack = go back up after each leaf.

Universal Backtracking Template

  1. Base case: solution is complete → add to results
  2. For each choice: try it (modify state)
  3. Recurse deeper
  4. Undo the choice (restore state)
  5. Prune: skip invalid choices early

When to Use Backtracking

  • Find ALL solutions (not just one)
  • Combinatorial problems (permutations, subsets)
  • Constraint satisfaction (N-Queens, Sudoku)
  • Path-finding in grids (Word Search)
  • Partitioning problems
// Universal backtracking skeleton
void Backtrack(State current, List<Solution> results)
{
    if (IsComplete(current))
    {
        results.Add(new Solution(current)); // Deep copy if needed!
        return;
    }

    foreach (var choice in GetChoices(current))
    {
        if (!IsValid(current, choice)) continue; // Prune!

        ApplyChoice(current, choice);          // Make move
        Backtrack(current, results);           // Recurse
        UndoChoice(current, choice);           // Undo move (backtrack)
    }
}

2. Subsets (Power Set)

Generate all 2ⁿ subsets of a given set. Each element is either included or excluded.

nums = [1, 2, 3] → 2³ = 8 subsets At each step: include current element OR skip it [] ← start ├── include 1: [1] │ ├── include 2: [1,2] │ │ ├── include 3: [1,2,3] ✓ │ │ └── skip 3: [1,2] ✓ │ └── skip 2: [1] │ ├── include 3: [1,3] ✓ │ └── skip 3: [1] ✓ └── skip 1: [] ├── include 2: [2] │ ├── include 3: [2,3] ✓ │ └── skip 3: [2] ✓ └── skip 2: [] ├── include 3: [3] ✓ └── skip 3: [] ✓
public IList<IList<int>> Subsets(int[] nums)
{
    var result = new List<IList<int>>();
    Backtrack(nums, 0, new List<int>(), result);
    return result;
}

private void Backtrack(int[] nums, int start, List<int> current, List<IList<int>> result)
{
    result.Add(new List<int>(current)); // Add snapshot (every node is a valid subset)

    for (int i = start; i < nums.Length; i++)
    {
        current.Add(nums[i]);              // Choose: include nums[i]
        Backtrack(nums, i + 1, current, result); // Recurse: only go forward
        current.RemoveAt(current.Count - 1); // Undo choice
    }
}
// Time: O(n * 2ⁿ) — 2ⁿ subsets, each takes O(n) to copy
// Space: O(n) recursion depth
// Key: start index prevents duplicates (only look at elements after current)

Subsets II — With Duplicates

public IList<IList<int>> SubsetsWithDup(int[] nums)
{
    Array.Sort(nums); // Sort to group duplicates together
    var result = new List<IList<int>>();
    Backtrack(nums, 0, new List<int>(), result);
    return result;
}

private void Backtrack(int[] nums, int start, List<int> current, List<IList<int>> result)
{
    result.Add(new List<int>(current));

    for (int i = start; i < nums.Length; i++)
    {
        // Skip duplicates at the same recursion level
        if (i > start && nums[i] == nums[i - 1]) continue;

        current.Add(nums[i]);
        Backtrack(nums, i + 1, current, result);
        current.RemoveAt(current.Count - 1);
    }
}

3. Combinations

Choose exactly k elements from n numbers (1 to n), in any order. Unlike subsets, all results have the same size k.

public IList<IList<int>> Combine(int n, int k)
{
    var result = new List<IList<int>>();
    Backtrack(1, n, k, new List<int>(), result);
    return result;
}

private void Backtrack(int start, int n, int k, List<int> current, List<IList<int>> result)
{
    if (current.Count == k)
    {
        result.Add(new List<int>(current));
        return;
    }

    // Pruning: remaining slots needed = k - current.Count
    // Last valid start = n - (k - current.Count) + 1
    int remaining = k - current.Count;
    for (int i = start; i <= n - remaining + 1; i++) // Pruning!
    {
        current.Add(i);
        Backtrack(i + 1, n, k, current, result);
        current.RemoveAt(current.Count - 1);
    }
}
// Time: O(C(n,k) * k), Space: O(k)
// Pruning: if start > n - remaining + 1, can't fill k elements anymore

Combination Sum — Unlimited Reuse (LeetCode 39)

Find all combinations from candidates that sum to target. Each number can be used unlimited times.

public IList<IList<int>> CombinationSum(int[] candidates, int target)
{
    Array.Sort(candidates); // Sort for pruning
    var result = new List<IList<int>>();
    Backtrack(candidates, 0, target, new List<int>(), result);
    return result;
}

private void Backtrack(int[] cands, int start, int remaining,
                         List<int> current, List<IList<int>> result)
{
    if (remaining == 0) { result.Add(new List<int>(current)); return; }

    for (int i = start; i < cands.Length; i++)
    {
        if (cands[i] > remaining) break; // Sorted → all further elements too big

        current.Add(cands[i]);
        Backtrack(cands, i, remaining - cands[i], current, result); // i (not i+1): reuse allowed
        current.RemoveAt(current.Count - 1);
    }
}

4. Permutations

Generate all n! orderings of a given array. Every element appears in every position exactly once across all permutations.

[1,2,3] → 3! = 6 permutations Strategy: at each step, pick any UNUSED element Track which elements are used with a boolean array Recursion depth = n, branching factor shrinks each level: Level 0: 3 choices (1, 2, or 3) Level 1: 2 choices (any unused) Level 2: 1 choice (last unused) Level 3: base case → add permutation Total leaves = 3 * 2 * 1 = 6 ✓
public IList<IList<int>> Permute(int[] nums)
{
    var result = new List<IList<int>>();
    Backtrack(nums, new bool[nums.Length], new List<int>(), result);
    return result;
}

private void Backtrack(int[] nums, bool[] used, List<int> current,
                         List<IList<int>> result)
{
    if (current.Count == nums.Length)
    {
        result.Add(new List<int>(current));
        return;
    }

    for (int i = 0; i < nums.Length; i++)
    {
        if (used[i]) continue;      // Already in current path

        used[i] = true;
        current.Add(nums[i]);
        Backtrack(nums, used, current, result);
        current.RemoveAt(current.Count - 1);
        used[i] = false;
    }
}
// Time: O(n * n!), Space: O(n)

Permutations II — With Duplicates

public IList<IList<int>> PermuteUnique(int[] nums)
{
    Array.Sort(nums);
    var result = new List<IList<int>>();
    Backtrack(nums, new bool[nums.Length], new List<int>(), result);
    return result;
}

private void Backtrack(int[] nums, bool[] used, List<int> current,
                         List<IList<int>> result)
{
    if (current.Count == nums.Length) { result.Add(new List<int>(current)); return; }

    for (int i = 0; i < nums.Length; i++)
    {
        if (used[i]) continue;
        // Skip: same value as previous, and previous was NOT used in this path
        // (if previous WAS used, this is a different branch — allow it)
        if (i > 0 && nums[i] == nums[i - 1] && !used[i - 1]) continue;

        used[i] = true;
        current.Add(nums[i]);
        Backtrack(nums, used, current, result);
        current.RemoveAt(current.Count - 1);
        used[i] = false;
    }
}

5. N-Queens — Classic Constraint Satisfaction

Place N queens on an N×N chessboard so no two queens attack each other (no two in same row, column, or diagonal).

N=4 solution (one of 2 solutions): . Q . . Row 0: queen at col 1 . . . Q Row 1: queen at col 3 Q . . . Row 2: queen at col 0 . . Q . Row 3: queen at col 2 Constraint check for row r, col c: - No queen in same column: colUsed[c] == false - No queen on same diagonal: (r-c) not in diag1Set ← top-left to bottom-right - No queen on same anti-diag: (r+c) not in diag2Set ← top-right to bottom-left
public IList<IList<string>> SolveNQueens(int n)
{
    var result = new List<IList<string>>();
    int[] queens = new int[n]; // queens[row] = column of queen in that row
    var cols   = new HashSet<int>();
    var diag1  = new HashSet<int>(); // row - col (same for top-left diagonal)
    var diag2  = new HashSet<int>(); // row + col (same for anti-diagonal)

    Backtrack(0, n, queens, cols, diag1, diag2, result);
    return result;
}

private void Backtrack(int row, int n, int[] queens,
    HashSet<int> cols, HashSet<int> diag1, HashSet<int> diag2,
    List<IList<string>> result)
{
    if (row == n)
    {
        result.Add(BuildBoard(queens, n));
        return;
    }

    for (int col = 0; col < n; col++)
    {
        if (cols.Contains(col))        continue; // Column taken
        if (diag1.Contains(row - col)) continue; // Diagonal taken
        if (diag2.Contains(row + col)) continue; // Anti-diagonal taken

        queens[row] = col;
        cols.Add(col); diag1.Add(row - col); diag2.Add(row + col);

        Backtrack(row + 1, n, queens, cols, diag1, diag2, result);

        cols.Remove(col); diag1.Remove(row - col); diag2.Remove(row + col);
    }
}

private List<string> BuildBoard(int[] queens, int n)
{
    var board = new List<string>();
    for (int r = 0; r < n; r++)
    {
        char[] row = new char[n];
        Array.Fill(row, '.');
        row[queens[r]] = 'Q';
        board.Add(new string(row));
    }
    return board;
}
// Time: O(n!), Space: O(n)

6. Sudoku Solver

Fill a 9×9 board with digits 1-9 such that each row, column, and 3×3 box contains each digit exactly once.

public void SolveSudoku(char[][] board)
{
    Solve(board);
}

private bool Solve(char[][] board)
{
    for (int r = 0; r < 9; r++)
    {
        for (int c = 0; c < 9; c++)
        {
            if (board[r][c] != '.') continue; // Already filled

            for (char digit = '1'; digit <= '9'; digit++)
            {
                if (!IsValid(board, r, c, digit)) continue;

                board[r][c] = digit;                 // Place digit
                if (Solve(board)) return true;    // Solution found
                board[r][c] = '.';                 // Backtrack
            }
            return false; // No valid digit → backtrack further
        }
    }
    return true; // All cells filled!
}

private bool IsValid(char[][] board, int row, int col, char digit)
{
    int boxRow = (row / 3) * 3, boxCol = (col / 3) * 3;

    for (int i = 0; i < 9; i++)
    {
        if (board[row][i] == digit) return false;        // Same row
        if (board[i][col] == digit) return false;        // Same column
        if (board[boxRow + i / 3][boxCol + i % 3] == digit) return false; // Same box
    }
    return true;
}
// Time: O(9^(empty cells)), Space: O(81) = O(1)

7. Word Search in a Grid

Given a 2D grid of characters and a word, return true if the word exists as a path of adjacent cells (horizontally or vertically). Each cell can only be used once per path.

Grid: Word: "ABCCED" A B C E S F C S A D E E Path: A(0,0)→B(0,1)→C(0,2)→C(1,2)→E(2,2)→D(2,1) ✓ Strategy: DFS from every cell that matches first letter. Mark cell as visited during DFS, unmark when backtracking.
public bool Exist(char[][] board, string word)
{
    int m = board.Length, n = board[0].Length;

    for (int r = 0; r < m; r++)
        for (int c = 0; c < n; c++)
            if (DFS(board, word, r, c, 0)) return true;

    return false;
}

private bool DFS(char[][] board, string word, int r, int c, int idx)
{
    if (idx == word.Length) return true;  // All chars matched!
    if (r < 0 || r >= board.Length || c < 0 || c >= board[0].Length) return false;
    if (board[r][c] != word[idx]) return false;

    char temp = board[r][c];
    board[r][c] = '#'; // Mark visited (in-place, no extra visited array)

    bool found = DFS(board, word, r + 1, c, idx + 1)
              || DFS(board, word, r - 1, c, idx + 1)
              || DFS(board, word, r, c + 1, idx + 1)
              || DFS(board, word, r, c - 1, idx + 1);

    board[r][c] = temp; // Restore (backtrack)
    return found;
}
// Time: O(m*n * 4^L) where L = word length. Space: O(L) recursion depth

8. Letter Combinations of a Phone Number

Map digits to letters (like phone keypad). Return all possible letter combinations a string of digits could represent.

public IList<string> LetterCombinations(string digits)
{
    if (digits.Length == 0) return new List<string>();

    string[] map = { "", "", "abc", "def", "ghi", "jkl", "mno", "pqrs", "tuv", "wxyz" };
    var result = new List<string>();
    Backtrack(digits, 0, new StringBuilder(), map, result);
    return result;
}

private void Backtrack(string digits, int idx, StringBuilder current,
                         string[] map, List<string> result)
{
    if (idx == digits.Length) { result.Add(current.ToString()); return; }

    foreach (char c in map[digits[idx] - '0'])
    {
        current.Append(c);
        Backtrack(digits, idx + 1, current, map, result);
        current.Remove(current.Length - 1, 1); // Backtrack
    }
}
// Time: O(4^n * n), Space: O(n)
// StringBuilder is faster than string concatenation for backtracking

9. Palindrome Partitioning

Partition a string so that every substring is a palindrome. Return all possible ways.

public IList<IList<string>> Partition(string s)
{
    var result = new List<IList<string>>();
    Backtrack(s, 0, new List<string>(), result);
    return result;
}

private void Backtrack(string s, int start, List<string> current,
                         List<IList<string>> result)
{
    if (start == s.Length) { result.Add(new List<string>(current)); return; }

    for (int end = start; end < s.Length; end++)
    {
        if (!IsPalindrome(s, start, end)) continue; // Prune non-palindromes

        current.Add(s.Substring(start, end - start + 1));
        Backtrack(s, end + 1, current, result);
        current.RemoveAt(current.Count - 1);
    }
}

private bool IsPalindrome(string s, int lo, int hi)
{
    while (lo < hi) { if (s[lo++] != s[hi--]) return false; }
    return true;
}
// Optimization: precompute palindrome DP table to check isPalindrome in O(1)

10. Pruning Strategies & Complexity

Backtracking without pruning explores the full search tree. Good pruning can reduce time dramatically — sometimes from O(n!) to feasible.

Common Pruning Techniques

  • Constraint propagation: Check constraints before going deeper (N-Queens, Sudoku)
  • Sorted input + break: If candidates are sorted and current is too large, break early (Combination Sum)
  • Bounds check: If remaining slots < elements needed, stop (Combinations)
  • Duplicate skipping: Skip elements equal to previous at same level
  • DP memoization: Cache sub-problem results if structure allows

Key Mental Model: Every backtracking solution has the same three parts: Choose (pick a candidate), Explore (recurse), Un-choose (undo the choice). If you can write these three clearly, the rest is just filling in the constraints.
Common Mistake — Shallow Copy: When adding the current path to results, always deep copy it: result.Add(new List<int>(current)). If you add the reference directly, all results will point to the same (empty) list after backtracking completes.
