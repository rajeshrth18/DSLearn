Climbing Stairs Easy LC #70

You can climb 1 or 2 steps at a time. Given n steps, return the number of distinct ways to climb to the top.

Example: n=4 → 5 ways: {1,1,1,1}, {1,1,2}, {1,2,1}, {2,1,1}, {2,2}

Constraints: 1 ≤ n ≤ 45

Recurrence: ways(n) = ways(n-1) + ways(n-2) [same as Fibonacci] Base cases: ways(1) = 1, ways(2) = 2 Pattern: 1D sequential DP, O(1) space
public int ClimbStairs(int n)
{
    if (n <= 2) return n;
    int prev2 = 1, prev1 = 2;
    for (int i = 3; i <= n; i++)
    {
        int curr = prev1 + prev2;
        prev2 = prev1; prev1 = curr;
    }
    return prev1;
}
// Time: O(n), Space: O(1)

Key Insight: To reach step n, you came from n-1 (one step) OR n-2 (two steps). This recurrence is identical to Fibonacci.

Min Cost Climbing Stairs Easy LC #746

Each step has a cost. You can start at index 0 or 1. Climb 1 or 2 steps at a time. Reach the top (past last step) with minimum cost.

Example: cost=[10,15,20] → 15 (step 1 → top)

dp[i] = min cost to REACH step i (pay cost[i] to LEAVE it) dp[i] = cost[i] + min(dp[i-1], dp[i-2]) Answer = min(dp[n-1], dp[n-2]) — jump to top from last or second-to-last
public int MinCostClimbingStairs(int[] cost)
{
    int n = cost.Length;
    int prev2 = cost[0], prev1 = cost[1];
    for (int i = 2; i < n; i++)
    {
        int curr = cost[i] + Math.Min(prev1, prev2);
        prev2 = prev1; prev1 = curr;
    }
    return Math.Min(prev1, prev2);
}
// Time: O(n), Space: O(1)

Maximum Subarray Easy LC #53

Find the contiguous subarray with the largest sum. Return the sum.

Example: [-2,1,-3,4,-1,2,1,-5,4] → 6 (subarray [4,-1,2,1])

Kadane's Algorithm: dp[i] = max subarray sum ENDING at index i dp[i] = max(nums[i], dp[i-1] + nums[i]) = either start fresh at i, OR extend previous subarray i: -2 1 -3 4 -1 2 1 -5 4 dp: -2 1 -2 4 3 5 6 1 5 maxSum = 6
public int MaxSubArray(int[] nums)
{
    int curr = nums[0], maxSum = nums[0];
    for (int i = 1; i < nums.Length; i++)
    {
        curr = Math.Max(nums[i], curr + nums[i]);
        maxSum = Math.Max(maxSum, curr);
    }
    return maxSum;
}
// Time: O(n), Space: O(1) — Kadane's Algorithm

Best Time to Buy and Sell Stock Easy LC #121

Given prices array, find the max profit from ONE buy-sell transaction. You must buy before you sell.

Example: [7,1,5,3,6,4] → 5 (buy at 1, sell at 6)

At each day, either we saw a new minimum price, or we check if selling today (prices[i] - minSoFar) gives a better profit. prices: 7 1 5 3 6 4 minSoFar: 7 1 1 1 1 1 profit: 0 0 4 2 5 3 ← max = 5
public int MaxProfit(int[] prices)
{
    int minPrice = int.MaxValue, maxProfit = 0;
    foreach (int price in prices)
    {
        minPrice = Math.Min(minPrice, price);
        maxProfit = Math.Max(maxProfit, price - minPrice);
    }
    return maxProfit;
}
// Time: O(n), Space: O(1)

DP View: dp[i] = max profit selling on day i = prices[i] - min(prices[0..i]).

House Robber Medium LC #198

You cannot rob two adjacent houses. Find the maximum money you can rob.

Example: [2,7,9,3,1] → 12 (rob houses 0, 2, 4)

dp[i] = max money robbing houses 0..i dp[i] = max(dp[i-1], // skip house i dp[i-2] + nums[i]) // rob house i nums: 2 7 9 3 1 dp: 2 7 11 11 12 → answer = 12
public int Rob(int[] nums)
{
    int n = nums.Length;
    if (n == 1) return nums[0];
    int prev2 = nums[0], prev1 = Math.Max(nums[0], nums[1]);
    for (int i = 2; i < n; i++)
    {
        int curr = Math.Max(prev1, prev2 + nums[i]);
        prev2 = prev1; prev1 = curr;
    }
    return prev1;
}
// Time: O(n), Space: O(1)

Coin Change Medium LC #322

Given coin denominations (infinite supply) and a target amount, return the fewest coins needed. If impossible, return -1.

Example: coins=[1,5,6,9], amount=11 → 2 (5+6)

dp[i] = min coins to make amount i dp[0] = 0 (base: 0 coins for amount 0) dp[i] = 1 + min(dp[i - coin]) for all coins ≤ i Unbounded knapsack: each coin usable multiple times
public int CoinChange(int[] coins, int amount)
{
    int[] dp = new int[amount + 1];
    Array.Fill(dp, amount + 1); // "infinity"
    dp[0] = 0;
    for (int i = 1; i <= amount; i++)
        foreach (int coin in coins)
            if (coin <= i)
                dp[i] = Math.Min(dp[i], 1 + dp[i - coin]);
    return dp[amount] > amount ? -1 : dp[amount];
}
// Time: O(amount * coins), Space: O(amount)

Longest Common Subsequence Medium LC #1143

Given two strings, return the length of their longest common subsequence (characters need not be contiguous).

Example: text1="abcde", text2="ace" → 3 ("ace")

dp[i][j] = LCS of text1[0..i-1], text2[0..j-1] if text1[i-1] == text2[j-1]: dp[i][j] = dp[i-1][j-1] + 1 else: dp[i][j] = max(dp[i-1][j], dp[i][j-1]) "" a c e "" 0 0 0 0 a 0 1 1 1 b 0 1 1 1 c 0 1 2 2 d 0 1 2 2 e 0 1 2 3 ← answer
public int LongestCommonSubsequence(string t1, string t2)
{
    int m = t1.Length, n = t2.Length;
    int[,] dp = new int[m + 1, n + 1];
    for (int i = 1; i <= m; i++)
        for (int j = 1; j <= n; j++)
            dp[i, j] = t1[i - 1] == t2[j - 1]
                ? dp[i - 1, j - 1] + 1
                : Math.Max(dp[i - 1, j], dp[i, j - 1]);
    return dp[m, n];
}
// Time: O(m*n), Space: O(m*n)

Word Break Medium LC #139

Given string s and a word dictionary, return true if s can be segmented into dictionary words.

Example: s="leetcode", dict=["leet","code"] → true

dp[i] = can we form s[0..i-1] using dictionary words? dp[0] = true (empty string) dp[i] = true if EXISTS j: dp[j]==true AND s[j..i-1] in dict s = "leetcode" dp[0]=T, dp[4]=T (dp[0] + "leet"), dp[8]=T (dp[4] + "code")
public bool WordBreak(string s, IList<string> wordDict)
{
    int n = s.Length;
    var set = new HashSet<string>(wordDict);
    bool[] dp = new bool[n + 1];
    dp[0] = true;
    for (int i = 1; i <= n; i++)
        for (int j = 0; j < i; j++)
            if (dp[j] && set.Contains(s.Substring(j, i - j)))
            { dp[i] = true; break; }
    return dp[n];
}
// Time: O(n²*L), Space: O(n)

Unique Paths Medium LC #62

A robot on an m×n grid can only move right or down. How many unique paths from top-left to bottom-right?

Example: m=3, n=7 → 28

dp[i][j] = paths to reach cell (i,j) dp[0][j] = 1 (top row: only way is go right) dp[i][0] = 1 (left col: only way is go down) dp[i][j] = dp[i-1][j] + dp[i][j-1]
public int UniquePaths(int m, int n)
{
    int[] dp = new int[n];
    Array.Fill(dp, 1);
    for (int i = 1; i < m; i++)
        for (int j = 1; j < n; j++)
            dp[j] += dp[j - 1];
    return dp[n - 1];
}
// Time: O(m*n), Space: O(n)

Partition Equal Subset Sum Medium LC #416

Given an integer array, determine if it can be partitioned into two subsets with equal sum.

Example: [1,5,11,5] → true (subsets [1,5,5] and [11])

Reduce to: can we pick items summing to total/2? → 0/1 Knapsack decision problem dp[w] = can we achieve sum w with items seen so far? Iterate capacity RIGHT-TO-LEFT (each item used at most once)
public bool CanPartition(int[] nums)
{
    int total = nums.Sum();
    if (total % 2 != 0) return false;
    int target = total / 2;
    bool[] dp = new bool[target + 1];
    dp[0] = true;
    foreach (int num in nums)
    {
        for (int w = target; w >= num; w--)
            if (dp[w - num]) dp[w] = true;
        if (dp[target]) return true;
    }
    return dp[target];
}
// Time: O(n * target), Space: O(target)

Longest Increasing Subsequence Medium LC #300

Return the length of the longest strictly increasing subsequence.

Example: [10,9,2,5,3,7,101,18] → 4 (subsequence [2,3,7,101])

O(n²) DP: dp[i] = length of LIS ending at index i dp[i] = 1 + max(dp[j]) for all j<i where nums[j]<nums[i] O(n log n) Patience Sorting: Maintain tails[] — smallest tail for LIS of each length Binary search to find where to insert each number
// O(n log n) solution:
public int LengthOfLIS(int[] nums)
{
    var tails = new List<int>();
    foreach (int num in nums)
    {
        int lo = 0, hi = tails.Count;
        while (lo < hi)
        {
            int mid = (lo + hi) / 2;
            if (tails[mid] < num) lo = mid + 1;
            else hi = mid;
        }
        if (lo == tails.Count) tails.Add(num);
        else tails[lo] = num;
    }
    return tails.Count;
}
// Time: O(n log n), Space: O(n)

Edit Distance Hard LC #72

Return the minimum number of operations (insert, delete, replace) to convert word1 to word2.

Example: word1="horse", word2="ros" → 3

dp[i][j] = min ops to convert word1[0..i-1] to word2[0..j-1] Base: dp[i][0]=i (delete all), dp[0][j]=j (insert all) if match: dp[i][j] = dp[i-1][j-1] else: dp[i][j] = 1 + min(replace, delete, insert) dp[i-1][j-1], dp[i-1][j], dp[i][j-1]
public int MinDistance(string word1, string word2)
{
    int m = word1.Length, n = word2.Length;
    int[,] dp = new int[m + 1, n + 1];
    for (int i = 0; i <= m; i++) dp[i, 0] = i;
    for (int j = 0; j <= n; j++) dp[0, j] = j;
    for (int i = 1; i <= m; i++)
        for (int j = 1; j <= n; j++)
            dp[i, j] = word1[i - 1] == word2[j - 1]
                ? dp[i - 1, j - 1]
                : 1 + Math.Min(dp[i - 1, j - 1],
                         Math.Min(dp[i - 1, j], dp[i, j - 1]));
    return dp[m, n];
}
// Time: O(m*n), Space: O(m*n)

Burst Balloons Hard LC #312

Burst all balloons to maximize coins. Bursting balloon i earns nums[i-1]*nums[i]*nums[i+1] (edges treated as 1).

Example: [3,1,5,8] → 167

Key Insight: Think "which balloon is burst LAST in range (i,j)" Add virtual balloons: [1, 3, 1, 5, 8, 1] dp[i][j] = max coins from open interval (i, j) For last balloon k in (i,j): coins = balloons[i]*balloons[k]*balloons[j] + dp[i][k] + dp[k][j] Why last? Because when k is burst last, i and j are still present as neighbors, so we know exactly what coins k earns.
public int MaxCoins(int[] nums)
{
    int n = nums.Length;
    int[] b = new int[n + 2];
    b[0] = b[n + 1] = 1;
    for (int i = 0; i < n; i++) b[i + 1] = nums[i];
    int N = n + 2;
    int[,] dp = new int[N, N];
    for (int len = 2; len < N; len++)
        for (int i = 0; i < N - len; i++)
        {
            int j = i + len;
            for (int k = i + 1; k < j; k++)
                dp[i, j] = Math.Max(dp[i, j],
                    b[i]*b[k]*b[j] + dp[i, k] + dp[k, j]);
        }
    return dp[0, N - 1];
}
// Time: O(n³), Space: O(n²)

Best Time to Buy and Sell Stock with Cooldown Hard LC #309

Multiple transactions allowed, but after selling you must rest 1 day (cooldown). Find max profit.

Example: [1,2,3,0,2] → 3 (buy@1, sell@3, cooldown, buy@0, sell@2)

State machine DP with 3 states: held = max profit while holding a stock sold = max profit on day just sold (must rest next day) rest = max profit during rest/idle (can buy next day) Transitions: held' = max(held, rest - price) // keep holding OR buy from rest sold' = held + price // sell what we hold rest' = max(rest, sold) // idle OR coming out of cooldown Initial: held = -INF (don't hold), sold = 0, rest = 0
public int MaxProfitCooldown(int[] prices)
{
    int held = int.MinValue / 2, sold = 0, rest = 0;

    foreach (int price in prices)
    {
        int prevHeld = held, prevSold = sold, prevRest = rest;
        held = Math.Max(prevHeld, prevRest - price);
        sold = prevHeld + price;
        rest = Math.Max(prevRest, prevSold);
    }
    return Math.Max(sold, rest);
}
// Time: O(n), Space: O(1) — state machine DP

Key Insight: Model the problem as a finite state machine. The 3 states capture all possible situations at any given day.

Regular Expression Matching Hard LC #10

Implement regex matching with . (any single char) and * (zero or more of preceding). Return true if pattern matches entire string.

Example: s="aab", p="c*a*b" → true ("c*" = zero c's, "a*" = two a's)

dp[i][j] = does s[0..i-1] match p[0..j-1]? Base: dp[0][0]=true, dp[0][j]: depends on '*' patterns Cases: 1) p[j-1] is letter or '.': dp[i][j] = dp[i-1][j-1] AND (s[i-1]==p[j-1] OR p[j-1]=='.') 2) p[j-1] is '*': a) Zero occurrences of p[j-2]: dp[i][j] = dp[i][j-2] b) One+ occurrences: dp[i][j] = dp[i-1][j] IF s[i-1] matches p[j-2] (letter or '.')
public bool IsMatch(string s, string p)
{
    int m = s.Length, n = p.Length;
    bool[,] dp = new bool[m + 1, n + 1];
    dp[0, 0] = true;

    // Handle patterns like "a*b*c*" matching empty string
    for (int j = 2; j <= n; j++)
        if (p[j - 1] == '*') dp[0, j] = dp[0, j - 2];

    for (int i = 1; i <= m; i++)
    {
        for (int j = 1; j <= n; j++)
        {
            if (p[j - 1] == '*')
            {
                // Zero occurrences of p[j-2]
                dp[i, j] = dp[i, j - 2];
                // One or more: current char matches p[j-2]
                if (p[j - 2] == '.' || p[j - 2] == s[i - 1])
                    dp[i, j] |= dp[i - 1, j];
            }
            else
            {
                // Direct match or '.' wildcard
                if (p[j - 1] == '.' || p[j - 1] == s[i - 1])
                    dp[i, j] = dp[i - 1, j - 1];
            }
        }
    }
    return dp[m, n];
}
// Time: O(m*n), Space: O(m*n)

Key Insight: The * wildcard is the only tricky case. It either means zero occurrences (skip the x* pair: dp[i][j-2]) or one-or-more (the current char must match, then extend: dp[i-1][j]).
